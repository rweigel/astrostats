2 Definitions

In the experiments done in the homework, the X i X_i Xi​ values are random samples from a probability distribution with expectation values for the mean and variance defined by E [ X ] ≡ μ E[X]\equiv\mu E[X]≡μ and E [ ( X − μ ) 2 ] ≡ σ 2 E[(X-\mu)^2]\equiv\sigma^2 E[(X−μ)2]≡σ2.

The probability distribution of the X ‾ \overline{X} Xs, each computed using n n n samples, is called the sampling distribution.

  • Population - “All” of the data is called the population. A population can be finite or infinite. An example of a finite population is all US citizens. An example of an infinite population are values from a continuous probability density function. (Technically, when you draw a value from a continuous probability distribution, you are actually drawing from a discrete distribution, because there are finite number of values that can be represented as 64–bit floating point numbers.)

  • Sample statistic - a computation based on a sample from a population that gives an estimate of the equivalent value that would be obtained if the same computation was performed on the population (see also Devore p214).

  • Population statistic - a computation based on all values in a population.

  • Probability distribution – There are two types: discrete and continuous.

  • Sampling distribution - When we take a sample from a population and compute a statistc, for example a mean, we want to know the uncertainty in the statistic. The probability histogram of the test statistic is the sampling distribution. If the sampling distribution is known, we can easily put error bars on our test statistic.

  • Expectation values

    The general definition of the expectation operator E [ ⋅ ] E[\cdot] E[⋅] is, for discrete x x x,

    E [ X ] = ∑ all  x x P ( x ) \displaystyle E \left[X\right] = \sum_{\text{all }x}xP(x) E[X]=all x∑​xP(x)

    or, for continuous x , x, x,

    E [ X ] = ∫ x x p ( x ) d x \displaystyle E\left[X\right] = \int_{x}xp(x)dx E[X]=∫x​xp(x)dx

    By definition, μ = E [ X ] \mu=E\left[X\right] μ=E[X]. That is μ \mu μ is the average of X X X over the population. The variance of a population is defined as

    σ 2 = E [ ( X − μ ) 2 ] \displaystyle \sigma^2=E\left[(X-\mu)^2\right] σ2=E[(X−μ)2]

    When we invent a statistic (a quantity computed from a sample) that is intended to be used as an estimate of a similar quantity from a population, we want the statistic to be unbiased. That is, the expectation value of the statistic should equal the population value. Two statistics that were considered are the mean and variance.

  • Random samples By random samples, we mean a value drawn from a population with a certain probability distribution. In general random samples implies “independent and identically distributed random variables” (often abbreviated iid).

  • Law of Large Numbers

    The weak law of large numbers [Rozanov, p69]: given an arbitrarily small δ > 0 \delta > 0 δ>0 and ϵ > 0 \epsilon > 0 ϵ>0, there is an integer n n n for which the quantity X ‾ ≡ ( 1 / n ) ( X 1 + … + X n ) \overline{X}\equiv (1/n)(X_1+…+X_n) X≡(1/n)(X1​+…+Xn​) (with X X X iid) will be in a small window centered on μ \mu μ with a probability near 1, that is,

    μ − ϵ ≤ X ‾ ≤ μ + ϵ \displaystyle \mu-\epsilon\le \overline{X} \le \mu + \epsilon μ−ϵ≤X≤μ+ϵ

    with a probability greater than 1 − δ 1-\delta 1−δ. This means we can choose very small ϵ \epsilon ϵ and δ \delta δ values and there will always be a value of n n n for which the constraints are satisfied.

    It follows from The Strong Law of Large Numbers that lim ⁡ n → ∞ X ‾ \lim_{n\rightarrow\infty}\overline{X} limn→∞​X exists and is equal to μ \mu μ with a probability of 1 [Rosanov, p 70]. (The Strong Law may seem equivalent to the Weak Law, but their statements are not quite equivalent.)

  • Central Limit Theorem

    The Law of Large Numbers tells use that if we require X ‾ \overline{X} X to fall in range that we specify around μ \mu μ with a probability that we specify, we can find an n n n value to satisfy our requirement. The central limit theorem says that for large n n n, X ‾ \overline{X} X is Gaussian-distributed with mean μ \mu μ and standard deviation σ \sigma σ. With the Central Limit theorem, we can make statements such as “I took a sample of n n n values and computed X ‾ \overline{X} X. If I took many samples and computed many X ‾ s \overline{X}s Xs, 95% of the time the range [ X ‾ − 1 . 9 6 σ / n , X ‾ + 1 . 9 6 σ / n ] [\overline{X}-1.96\sigma/\sqrt{n},\overline{X}+1.96\sigma/\sqrt{n}] [X−1.96σ/n​,X+1.96σ/n​] would include (“trap”) μ \mu μ.

  • Parametric bootstrap sample statistic distribution – A pmf or pdf created by simulating many experiments and the calculation of a sample statistic for each experiment. For example, suppose that we wanted to know the distribution of the sample statistic Y = X 1 2 + X 2 2 + … + X n 2 Y=X_1^2 + X_2^2 + … + X_n^2 Y=X12​+X22​+…+Xn2​, where an experiment consists of drawing n n n values, X X X, which are Gaussian distribution with a known mean and standard deviation. We could use a random number generator so simulate many experiements and for each experiment compute Y Y Y. If a theoretical distribution for Y Y Y is not known, this method can be used.

  • Non-parametric bootstrap sample statistic distribution

    The definition of the idiom “bootstrapping” is “get (oneself or something) into or out of a situation using existing resources.” If we don’t know the sampling distribution of X X X, we can’t use a random number generator to simulate many experiments. In this case, we use the only available resource: the measurments from one experiment. This done by drawing a sample of n n n with replacement from the n n n measurements. Each such draw is called a bootstrap sample (or experiment) and the test statistic computed from it is indicated by Y ∗ Y_* Y∗​. The bootstrap pmf or pdf is computed from the histogram of the Y ∗ Y^* Y∗s.

3 Key Distributions

Discrete distributions used thus far

  • The Binomial Distribution, which follows from Bernoulii Trials.

  • The Poisson Distribution, which is limiting case of Binomial Distribution (requires n ≫ 1 n\gg 1 n≫1 and valid for k p ≪ 1 kp\ll 1 kp≪1 and k / n ≪ 1 k/n\ll 1 k/n≪1) (did derivation in class).

  • The Exponential Distribution (aka “Waiting Time Distribution)”, which is the distribution of the time between successes (aka “events”) in a Poisson-distributed variable (derivation is not trivial).

  • The Normal or Gaussian distribution (note Normal and Gaussian are used interchangeably), which is a limiting case of Binomial Distribution (and Poisson Distribution, but not Exponential distribution).

Three key distributions are

  • The “Standard Normal” is the sampling distribution of the quantity

    z = X ‾ − μ σ / n \displaystyle z = \frac{\overline{X}-\mu}{\sigma/\sqrt{n}} z=σ/n​X−μ​

    Constraint: X X X is a random variable with mean μ \mu μ and standard deviation σ \sigma σ and n \boldsymbol{n} n is large.

    A standardized variable will have a histogram that is centered on the origin and a standard deviation of unity.

    If X X X is a random variable from almost any probability distribution with mean μ \mu μ and standard deviation σ \sigma σ, the sampling distribution of z z z is the Standard Normal.

  • The “Student t t t” distribution is the sampling distribution of the quantity

    t = X ‾ − μ S / n \displaystyle t = \frac{\overline{X}-\mu}{S/\sqrt{n}} t=S/n​X−μ​

    where

    S = 1 n − 1 ∑ i = 1 n ( X − X ‾ ) 2 \displaystyle S = \sqrt{\frac{1}{n-1}\sum_{i=1}^n (X-\overline{X})^2} S=n−11​i=1∑n​(X−X)2​

    Constraint: X X X is a Gaussian–distributed random variable. Both the numerator and denominator of t t t will vary from sample to sample and so we expect that the histogram of

    t = X ‾ − μ S / n \displaystyle t=\frac{\overline{X}-\mu}{S/\sqrt{n}} t=S/n​X−μ​

    to be “fatter” or have “fatter tails” than

    z = X ‾ − μ σ / n \displaystyle z=\frac{\overline{X}-\mu}{\sigma/\sqrt{n}} z=σ/n​X−μ​

    The t t t distribution is actually a family of distributions that depend on n n n and so “ t t t distribution” is ambiguous. We usually discuss “a t t t distribution with ν \nu ν” degrees of freedom, where ν ≡ n − 1 \nu \equiv n-1 ν≡n−1.

  • Often we do not know the sampling distribution; in the examples given above, there was a severe constraint on the distribution of X X X. In general, given an arbitrary distribution of X X X and an arbitrary statistic derived from it, we don’t know the sampling distribution of the statistic.

    We can use the boostrap method to derive a sampling distribution using data from a single sample.
