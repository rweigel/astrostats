The first two problems of this homework are intended to primarily be a review of the programming techniques that you will need for this class. The problems mention the Law of Large Numbers, confidence intervals, and hypothesis testing. You do not need to know anything about the statistical theory behind them in order to solve these problems.

The Law of Large Numbers tells us that as n → ∞ n\rightarrow \infty n→∞ the sample average defined by

X ‾ = 1 n ∑ i = 1 n X i \displaystyle \overline{X}=\frac{1}{n}\sum_{i=1}^n X_i X=n1​i=1∑n​Xi​

will be near the population average μ \mu μ with a given probability. Given n n n samples from a population, we don’t expect X ‾ \overline{X} X to exactly match μ \mu μ. The Law of Large Numbers allows us to make a statement about the difference X ‾ − μ \overline{X}-\mu X−μ. Specifically, the statement involves the probability that ∣ X ‾ − μ ∣ |\overline{X}-\mu| ∣X−μ∣ is smaller that a certain value.

If you are interested, for a more formal definition of the Law of Large Numbers and proofs, see

  • Orloff and Bloom, Reading 6b
  • Bulmer, Chapter 6 (Piazza)
  • DeGroot, Chapter 6 (Piazza)

Answer

In this problem, a single sample of n = 1 0 0 n=100 n=100 was used to compute an average, X ‾ o \overline{X}_o Xo​ and standard deviation s o s_o so​. We do not know the population average μ \mu μ but want to make a statement (or “inference”) about it.

Now do many (say 10,000) experiments of drawing a sample of 1 0 0 100 100 values from a gaussian distribution with mean X ‾ o \overline{X}_o Xo​ and sample standard deviation s o s_o so​. That is, assume that the actual unknown population distribution has a mean and standard deviation that is equal to that from the sample.

3.2 Bayes’ Rule Derivation and Terminology

You only need the law of multiplication to derive Bayes’ theorem. The law of multiplication is

P ( A  and  B ) = P ( A ) P ( B ∣ A ) \displaystyle P(A\text{ and }B)=P(A)P(B|A) P(A and B)=P(A)P(B∣A)

The labels A A A and B B B are arbitrary. Swapping them gives:

P ( B  and  A ) = P ( B ) P ( A ∣ B ) \displaystyle P(B\text{ and }A)=P(B)P(A|B) P(B and A)=P(B)P(A∣B)

P ( B  and  A ) P(B\text{ and }A) P(B and A) means the same thing as P ( A  and  B ) P(A\text{ and }B) P(A and B) (mathematically, the set intersection A  and  B A\text{ and }B A and B is commutative). Equating the above two equations gives

P ( B ) P ( A ∣ B ) = P ( A ) P ( B ∣ A ) \displaystyle P(B)P(A|B)=P(A)P(B|A) P(B)P(A∣B)=P(A)P(B∣A)

Bayes’ theorem (“theorem”, “rule”, and “law” are all used, seemingly arbitrarily) is

P ( A ∣ B ) = P ( B ∣ A ) P ( A ) P ( B ) \displaystyle P(A|B)=P(B|A)\frac{P(A)}{P(B)} P(A∣B)=P(B∣A)P(B)P(A)​

In this form,

  • P ( A ∣ B ) P(A|B) P(A∣B) is called the posterior. It is a probability that we compute after (post) consideration of the other probabilities.
  • P ( B ∣ A ) P(B|A) P(B∣A) is called the likelihood (this is a poor choice of name)
  • P ( A ) P(A) P(A) is called the prior. If B B B and A A A are independent, then P ( A ∣ B ) = P ( A ) P(A|B)=P(A) P(A∣B)=P(A), so P ( A ) P(A) P(A) is the probability prior to knowing any relationship between A A A and B B B.
  • P ( B ) P(B) P(B) is called a normalizing factor [Wall and Jenkins, p26] or, when B B B is evidence and A A A is a hypothesis, P ( B ) P(B) P(B) is also referred to as the evidence.

Another form of Bayes’ theorem, valid when the events A j A_j Aj​ are mutually exclusive and exhaustive, has the denominator re-written using the law of total probability, which is

P ( B ) = ∑ j = 1 k P ( A j ) P ( B ∣ A j ) . \displaystyle P(B) = \sum_{j=1}^kP(A_j)P(B|A_j). P(B)=j=1∑k​P(Aj​)P(B∣Aj​).

In this case, Bayes’ theorem is

P ( A j ∣ B ) = P ( B ∣ A j ) P ( A j ) ∑ j = 1 k P ( A j ) P ( B ∣ A j ) . \displaystyle P(A_j|B)=P(B|A_j)\frac{P(A_j)}{\sum_{j=1}^kP(A_j)P(B|A_j)}. P(Aj​∣B)=P(B∣Aj​)∑j=1k​P(Aj​)P(B∣Aj​)P(Aj​)​.
